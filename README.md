# Telegram Clone Client

This is the unofficial Telegram Clone web-client made with React.

P.S. Work in progress...

## Technologies

- React  
- React-Redux + Saga  
- Typescript  
- Socket.IO / WebSockets (in progress)  
- HTML  
- CSS
- JWT Authorization
